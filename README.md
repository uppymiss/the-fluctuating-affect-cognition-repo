# Research Study

Repository for our Researsch Project materials.

## Project (short description)
This project examines cycle-related variability in attention, impulsivity, and emotional regulation, with a focus on menstrual-cycle phases and cross-gender comparisons. It explores whether fluctuations in executive control and affect regulation align with standardized self-report measures.

- `/experiments` – experiment code
- `/data` – anonymized datasets
